Q1. Write a PHP script to accept a string and then display each word of string in reverse  
order by using self processing form 

short and easy without skipping any line

Here is a short, clean, complete PHP self-processing form (nothing skipped, includes HTML + PHP in one file):

<!DOCTYPE html>
<html>
<head>
    <title>Reverse Words</title>
</head>
<body>

<form method="post">
    Enter String:
    <input type="text" name="str" required>
    <input type="submit" name="submit" value="Reverse">
</form>

<?php
if(isset($_POST['submit']))
{
    $str = $_POST['str'];

    // Convert string into array of words
    $words = explode(" ", $str);

    // Reverse array
    $words = array_reverse($words);

    // Join words back into string
    $result = implode(" ", $words);

    echo "<br>Reversed String: " . $result;
}
?>

</body>
</html>
How it works (quick clarity):
explode() → breaks string into words

array_reverse() → reverses order

implode() → joins back into string

?>

</body>

Q16. Write a script to create XML file named “Item.xml” 
<Product> 
<ProductName>............................................................. </ProductName> 
<ProductPrice> ................................................................</ProductPrice> 
<Quantity>....................................................................</Quantity> 
</Product> 
Store the details of 5 Products of different Types 
Link “ Product. xml” file to the CSS style sheet and get well formatted output as given below 
ProductName : 
color : red; 
font-family : copperplate Gothic Light; font-size : 16pt; 
font :bold;. 

short and easy without skipping any line

Here is a short, complete solution (PHP to create XML + CSS file, nothing skipped):

1. PHP Script (create Item.xml)
<?php
// Create XML
$xml = new SimpleXMLElement('<?xml version="1.0"?><Products></Products>');

// Add 5 products
for($i = 1; $i <= 5; $i++)
{
    $p = $xml->addChild('Product');
    $p->addChild('ProductName', 'Product'.$i);
    $p->addChild('ProductPrice', rand(100,500));
    $p->addChild('Quantity', rand(1,10));
}

// Add CSS link
$dom = dom_import_simplexml($xml)->ownerDocument;
$dom->insertBefore(
    $dom->createProcessingInstruction('xml-stylesheet', 'type="text/css" href="style.css"'),
    $dom->firstChild
);

// Save file
$xml->asXML("Item.xml");

echo "Item.xml created successfully";
?>
2. CSS File (style.css)
ProductName {
    display: block;
    color: red;
    font-family: "Copperplate Gothic Light";
    font-size: 16pt;
    font-weight: bold;
}

ProductPrice, Quantity {
    display: block;
    margin-left: 20px;
}

Q22. Create the following tables PostgreSQL. 
Doctor (doc_no, dname, address ,city ,area) 
Hospital (hosp_no, hname, hcity) 
Doctor-Hospital related with many-one relationship. 
Create a RDB in 3NF for above and solve the following. 
Write a script in PHP to print the number of Doctors visiting to the Hospital in tabular 
format. Accept Hospital name from user.

short and easy without skipping any line

Here is a short and complete solution (no lines skipped):

1. PostgreSQL Tables (3NF)
CREATE TABLE Hospital (
    hosp_no SERIAL PRIMARY KEY,
    hname VARCHAR(50),
    hcity VARCHAR(50)
);

CREATE TABLE Doctor (
    doc_no SERIAL PRIMARY KEY,
    dname VARCHAR(50),
    address VARCHAR(100),
    city VARCHAR(50),
    area VARCHAR(50),
    hosp_no INT,
    FOREIGN KEY (hosp_no) REFERENCES Hospital(hosp_no)
);
2. PHP Program
<?php

$conn = pg_connect("host=localhost dbname=test user=postgres password=1234");

if(!$conn) {
    echo "Connection Failed";
    exit;
}

echo "Enter Hospital Name: ";
$name = trim(fgets(STDIN));

$query = "SELECT COUNT(*) FROM Doctor d, Hospital h 
          WHERE d.hosp_no = h.hosp_no AND h.hname = '$name'";

$result = pg_query($conn, $query);

$row = pg_fetch_row($result);

echo "Number of Doctors: ".$row[0];

pg_close($conn);

?>
Key Points (for exam)
Many-to-one → Doctor → Hospital

Foreign key → hosp_no

COUNT(*) used

Join used with hospital name
