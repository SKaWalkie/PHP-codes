Q1. Write a program to read a flat file “student.dat”, calculate the percentage and display the data 
from file in tabular format. (Student.dat file contains rollno, name, marks of OS, WT, DS, 
Python)

short and easy without skipping any line

<?php
$filename = "student.dat";

// check if file exists
if (file_exists($filename)) {

    $file = fopen($filename, "r");

    echo "<table border='1'>";
    echo "<tr>
            <th>Roll No</th>
            <th>Name</th>
            <th>OS</th>
            <th>WT</th>
            <th>DS</th>
            <th>Python</th>
            <th>Percentage</th>
          </tr>";

    // read file line by line
    while (($line = fgets($file)) != false) {

        // split data
        $data = explode(" ", trim($line));

        $roll = $data[0];
        $name = $data[1];
        $os = $data[2];
        $wt = $data[3];
        $ds = $data[4];
        $py = $data[5];

        // calculate percentage
        $total = $os + $wt + $ds + $py;
        $per = $total / 4;

        // display row
        echo "<tr>
                <td>$roll</td>
                <td>$name</td>
                <td>$os</td>
                <td>$wt</td>
                <td>$ds</td>
                <td>$py</td>
                <td>$per%</td>
              </tr>";
    }

    echo "</table>";

    fclose($file);

} else {
    echo "File not found!";
}
?>

Q3. Write a menu driven program to perform various file operations. Accept filename from 
user. 
a) Display type of file. 
b) Display last modification time of file. 
c) Display the size of file.

<?php

echo "Enter file name: ";
$fname = trim(fgets(STDIN));

// check file exists
if(!file_exists($fname)) {
    echo "File does not exist\n";
    exit;
}

do {
    echo "\n1. Display type of file";
    echo "\n2. Display last modification time";
    echo "\n3. Display file size";
    echo "\n4. Exit";
    echo "\nEnter choice: ";

    $ch = trim(fgets(STDIN));

    switch($ch) {

        case 1:
            if(is_file($fname))
                echo "It is a normal file\n";
            else if(is_dir($fname))
                echo "It is a directory\n";
            else
                echo "Unknown type\n";
            break;

        case 2:
            $time = filemtime($fname);
            echo "Last Modified: " . date("d-m-Y H:i:s", $time) . "\n";
            break;

        case 3:
            $size = filesize($fname);
            echo "File Size: " . $size . " bytes\n";
            break;

        case 4:
            echo "Exit\n";
            break;

        default:
            echo "Invalid choice\n";
    }

} while($ch != 4);

?>

$valid_pass = "1234";

    $_SESSION['count'] = 0;

CREATE TABLE Hospital (
    hosp_no SERIAL PRIMARY KEY,
    hname VARCHAR(50),
    hcity VARCHAR(50)
);

    doc_no SERIAL PRIMARY KEY,
    dname VARCHAR(50),

Q22. Create the following tables PostgreSQL. 
Doctor (doc_no, dname, address ,city ,area) 
Hospital (hosp_no, hname, hcity) 
Doctor-Hospital related with many-one relationship. 
Create a RDB in 3NF for above and solve the following. 
Write a script in PHP to print the number of Doctors visiting to the Hospital in tabular 
format. Accept Hospital name from user.

short and easy without skipping any line

Here is a short and complete solution (no lines skipped):

1. PostgreSQL Tables (3NF)
CREATE TABLE Hospital (
    hosp_no SERIAL PRIMARY KEY,
    hname VARCHAR(50),
    hcity VARCHAR(50)
);

CREATE TABLE Doctor (
    doc_no SERIAL PRIMARY KEY,
    dname VARCHAR(50),
    address VARCHAR(100),
    city VARCHAR(50),
    area VARCHAR(50),
    hosp_no INT,
    FOREIGN KEY (hosp_no) REFERENCES Hospital(hosp_no)
);
2. PHP Program
<?php

$conn = pg_connect("host=localhost dbname=test user=postgres password=1234");

if(!$conn) {
    echo "Connection Failed";
    exit;
}

echo "Enter Hospital Name: ";
$name = trim(fgets(STDIN));

$query = "SELECT COUNT(*) FROM Doctor d, Hospital h 
          WHERE d.hosp_no = h.hosp_no AND h.hname = '$name'";

$result = pg_query($conn, $query);

$row = pg_fetch_row($result);

echo "Number of Doctors: ".$row[0];

pg_close($conn);

?>
Key Points (for exam)
Many-to-one → Doctor → Hospital

Foreign key → hosp_no

COUNT(*) used

Join used with hospital name
